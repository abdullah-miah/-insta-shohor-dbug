How does javascript works ?

javaScript is high level programming language for web application. JavaScript make interactivity in web site . This code run browser site.Browser use some engin like chrome use: V8, Firefox : spidermonkey . JavaScript followed ECMA standard.

Session Storage vs Local Storage ?

Session Storage:

session storage store key and values. When user visit web site , web authority store some cookies . This session ends when close browser .

Local Storage:

Local storage store cookies in browser. Local storage data no expiration date. Local storage available alwayes. .

How does event loop works ?

Declar a event loop, then loop go web/API, c++ API, then go queue(FIFO). Queue and call stack through a loop . This loop called event loop. Event loops wait for call stack to be clear before pushing call backs from task Queue. Event loop maintain browser engine. Node js maintain JavaScript Event loop.
